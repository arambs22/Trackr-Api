# Trackr-Api

Backend API tipo Trello para manejar **Boards → Lists → Tasks**, con **JWT Auth**, **PostgreSQL** y **Swagger UI**.

## Stack
- Java 17
- Spring Boot
- Spring Security (JWT)
- PostgreSQL (Docker)
- Spring Data JPA
- Swagger / OpenAPI

## Requisitos
- Java 17
- Maven
- Docker Desktop
